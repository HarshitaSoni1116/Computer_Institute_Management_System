# TechVision Computer Institute Website

A full-stack Computer Institute Management Website developed using React.js, Spring Boot, and MySQL. The platform is designed for computer training institutes offering courses such as DCA, PGDCA, CPCT Preparation, Tally Prime, Python Programming, and Web Development.

## Features

* Modern Responsive User Interface
* Dark Green & Gold Premium Theme
* Course Management Section
* Student Testimonials
* Placement Partners Showcase
* Institute Gallery
* Free Learning Resources
* AI Chatbot Interface
* Google Maps Integration
* Admission Inquiry Form
* Spring Boot REST APIs
* MySQL Database Integration
* Smooth Scrolling Navigation
* Interactive Animations and Effects
* Component-Based Architecture

## Technology Stack

Frontend:

```bash

npm start
```

Backend:

```bash
mvn spring-boot:run
```

Database:

```sql
CREATE DATABASE computerClass;
```
